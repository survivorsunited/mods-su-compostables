# Compostables

## üå± About

Because it should be in real life! This mod makes composting more realistic by allowing you to compost organic items that logically should decompose, including rotten flesh, spider eyes, eggs, meat, fish, soil blocks, chorus fruits, stews, and even dyes made from organic materials.

## ‚ú® Features

- **100 New Compostable Items** - From dead bushes (10%) to raw meat, stews, dyes, carpets, wool, and bones (100%)
- **Balanced Progression** - Each item has carefully balanced composting chances
- **Villager Compatibility** - Farmer villagers will pick up and compost the new items
- **Server-Side Only** - No client installation required
- **Lightweight** - Minimal performance impact

## üßë‚Äçüåæ Farmer Villagers

Farmer villagers will now pick up and compost:
- All portable organic items (76 items excluding blocks)
- Raw and cooked meat/fish
- Chorus fruits and flowers
- Stews and soups
- Eggs and organic waste
- Dyes and carpets
- Bones and string

*Note: Farmers cannot pick up blocks (Grass Block, Dirt Path, Rooted Dirt, Podzol, Mycelium, Nylium blocks, Muddy Mangrove Roots, Bamboo Block, Bone Block, Wool blocks), but players can manually compost them.*
